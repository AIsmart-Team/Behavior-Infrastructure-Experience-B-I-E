# E-Taxi-Charging-Behavior–Infrastructure–Experience (B–I–E)
A behavior-aware analytical framework for modeling electric taxi charging patterns and evaluating charging infrastructure efficiency and service quality.

# Overview

This repository accompanies a city-scale empirical study on electric taxi (E-taxi)
charging systems, adopting an integrated
**Behavior–Infrastructure–Experience (B–I–E)** perspective.

Rather than focusing on charging demand or infrastructure in isolation,
the study explicitly links:
(1) heterogeneous charging behaviors,
(2) structural characteristics of charging infrastructure, and
(3) drivers’ experienced charging service outcomes.

---


## Components

The repository implements three main analytical components:

1. **Behavioral Regularity Characterization**  
   Charging regularity is quantified across temporal rhythms, spatial preferences,
   and charging strategies using a spatio-temporal–strategic entropy
   (STS-Entropy) framework.

2. **Infrastructure Diagnostics**  
   Two complementary indicators are used to assess structural adequacy:
   - Fast-Charging Mismatch Rate (FCMR), capturing misalignment between fast-charging
     demand and capacity.
   - Charging–Amenity Coordination Index (CACI), evaluating whether charging stations
     provide coordinated support for charging and drivers’ basic rest needs.

3. **Behavior–Service Coupling Assessment**  
   The Regularity–Service Accessibility Coupling (RSAC) index is used to examine
   whether more regular charging behavior systematically translates into improved
   charging service access under existing infrastructure configurations.
